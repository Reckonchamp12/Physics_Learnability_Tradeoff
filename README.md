> **GC-Benchmark** is a rigorous, reproducible benchmark comparing **15 ML models** across forward prediction and inverse design tasks for silicon photonic grating couplers, evaluated on **five physically distinct 50 k-sample datasets**. The repository also ships the dataset generator, four ablation variants, FEM validation tools, and a 10-test statistical validation suite.

## Overview

Grating couplers are critical interface components in silicon photonic integrated circuits. Their design involves a non-trivial mapping between five geometric parameters and the optical transmission spectrum.

| Task | Direction | Input | Output |
|------|-----------|-------|--------|
| **Forward prediction** | geometry → optics | 5 geometry params | 4 scalar metrics or 100-point spectrum |
| **Inverse design** | optics → geometry | 4 scalar metrics or 100-point spectrum | 5 geometry params |

We evaluate **15 distinct models** across **5 physically distinct datasets** generated by a physics-constrained semi-analytical model with controlled ablations of individual physics terms.

## Datasets and Generator

All datasets contain **N = 50,000** grating coupler simulations, **L = 100** wavelength points (1.2–1.6 µm, telecom C-band).

| Dataset | Ablation | Description |
|---------|----------|-------------|
| `Reference` | None | Full physics: energy conservation, Fabry-Pérot fringes, measurement noise |
| `A_NoEnergyConservation` | Energy normalisation | T + R + A may deviate from 1 |
| `B_NoFabryPerot` | FP fringes | Pure Lorentzian resonance, no cavity oscillations |
| `C_FixedBandwidth` | Bandwidth variability | Resonance width fixed at 40 nm for all samples |
| `D_NoNoise` | Measurement noise | Clean analytical spectra |

### Feature Space

**Geometry inputs** (5 parameters):

| Parameter | Range | Description |
|-----------|-------|-------------|
| `period_nm` | 300–700 nm | Grating pitch |
| `fill_factor` | 0.3–0.7 | Groove duty cycle |
| `etch_depth_nm` | 50–200 nm | Silicon etch depth |
| `oxide_thickness_nm` | 1000–2000 nm | Buried oxide thickness |
| `si_thickness_nm` | 200–300 nm | Silicon device layer |

**Scalar outputs** (4 metrics): `lambda_center_nm`, `bandwidth_um`, `n_eff`, `peak_transmission`

**Spectral output**: transmission T(λ) over 100 wavelength points.

### Data Splits

| Split | Fraction | N |
|-------|----------|---|
| Train | 70% | 35,000 |
| Val   | 15% | 7,500  |
| Test  | 15% | 7,500  |

## Physics Generator

`PhysicallyConsistentGratingGenerator` uses a semi-analytical waveguide model:

1. **Effective index** — slab confinement + grating modulation (linear EMT) + etch correction + oxide substrate effect.
2. **Coupling resonance** — Lorentzian centred at λ = Λ × n_eff, width from fill factor and etch depth.
3. **Fabry-Pérot fringes** — two sinusoidal terms from the waveguide round-trip length.
4. **Fresnel reflection** — R₀ from interface, geometry-modulated.
5. **Energy conservation** — enforced exactly by normalisation: R̃ = R/s, T̃ = T/s, Ã = A/s.
6. **Measurement noise** — Gaussian (σ = 0.01), re-normalised.

**Note on negative absorption**: Small negative A values (|A_min| < 0.007) arise from float32 arithmetic in the closure definition A = 1 − R − T combined with noise injection. Mean energy conservation error is below 10⁻⁸. These artefacts are intentionally retained to preserve spectral differentiability for ML training.

### Ablation Variants

| Class | What changes |
|-------|-------------|
| `AblationA_NoEnergyConservation` | Normalisation step omitted |
| `AblationB_NoFabryPerot` | FP terms zeroed |
| `AblationC_FixedBandwidth` | Bandwidth = 40 nm (constant) |
| `AblationD_NoNoise` | `add_noise=False` always |

## Dataset Validation

### Physical Validation (`PhysicallyCorrectValidator`)

| Check | Criterion |
|-------|-----------|
| Energy conservation | ⟨\|R+T+A−1\|⟩ < 0.01 |
| Physical bounds | R, T ∈ [0,1]; A may be slightly negative |
| Parameter coverage | Ranges within 5% of specification |

### Statistical Validation — 10 Tests (`StatisticalDatasetValidator`)

| Test | Criterion | Key result |
|------|-----------|------------|
| 1. Parameter independence | max\|ρ\| < 0.3 | Parameters independently sampled |
| 2. Geometry–physics monotonicity | ρ(period, λ_c) > 0.9 | Confirmed (ρ = 0.977) |
| 3. Spectral regularity | Mean first-diff < 0.05 | Spectra are smooth |
| 4. Dataset uniqueness | Duplicate rate < 1% | No near-duplicates |
| 5. Output-space entropy | > 70% of discrete grid maximum | Broad coverage |
| 6. Train-test KS equivalence | KS p > 0.05 | Splits statistically equivalent |
| 7. Noise robustness | Peak stability > 85% | Argmax stable under σ=0.01 noise |
| 8. PCA compressibility | 10 PCs explain > 90% variance | Low-dimensional spectrum manifold |
| 9. Physical bounds adherence | R, T ∈ [0,1] | No bound violations |
| 10. Parameter coverage | KS vs uniform: p > 0.01 | Uniformly distributed |

**Note on Test 5**: Normalised entropy > 100% indicates fine-scale sub-grid variability — a sign of excellent coverage, not an error. The test evaluates coverage and diversity, not Shannon-theoretic bounds.

**Note on Test 2**: Weak ρ(fill_factor, bandwidth) ≈ 0.03 is physically correct — bandwidth depends nonlinearly on multiple parameters, so weak pairwise correlation is expected.

### FEM Validation (`FEMValidator`)

Compares generator spectra against real FEM CSV outputs.

| Metric | Definition |
|--------|-----------|
| λ_c relative error (%) | \|λ_c^FEM − λ_c^gen\| / λ_c^FEM × 100 |
| Bandwidth relative error (%) | Second-moment bandwidth comparison |
| Pearson correlation | After peak-alignment via wavelength shift |
| FEM energy error | max\|T + R − 1\| (passivity check on FEM data) |

## Tasks and Metrics

### Forward — Scalar
R², MAE, RMSE, MAPE, Pearson correlation

### Forward — Spectrum
MSE, MAE, Cosine similarity, SAM (°), DTW distance, LSD (dB), Peak wavelength error, FWHM error

**Domain-specific metrics** (from learnability study):

| Metric | Formula | Interpretation |
|--------|---------|----------------|
| **RLE** | (1/N) Σ\|λ_peak_true − λ_peak_pred\| / FWHM_true | < 0.5 → error < half bandwidth |
| **ECE-ML** | E_λ\[\|R_est + T_pred − 1\|\] | Physical energy budget accuracy |
| **IPE** | (1/N) Σ ∫\|T_pred − T_true\| dλ [nm] | Spectrally integrated absolute error |

### Physics Consistency
Energy conservation violation, Fabry-Pérot ripple, Hilbert causality metric

### Inverse — Geometry
Success rate @ 1/2/5/10%, MAE, RMSE, median relative error, 90% coverage, multimodal rate, cycle-consistency error

### Key Implementation Notes

- **FNO1D**: `use_amp_override=False`. cuFFT fp16 requires power-of-two signal lengths; a 100-point grid violates this. `FNOLayer.forward()` also casts to fp32 as a safety net.
- **NeuralODE**: Uses fixed-step Euler (5 steps) if `torchdiffeq` is absent.
- **DTW**: Falls back to Euclidean distance if `dtaidistance` is absent.
- **cINN**: MAP estimate for latency; 100 samples for 90% coverage.
- **GP**: Trained on 2,000 samples; evaluated on full test set.
- **Negative absorption**: Bounded artefact from float32 closure (|A_min| < 0.007); not post-processed to preserve spectral differentiability.
- **Ablation study**: Model retraining is omitted from the ablation analysis to avoid conflating dataset quality with learner-specific inductive biases. Task-level performance is established on the full dataset separately.
